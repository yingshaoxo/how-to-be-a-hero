# I have super power

## 总构想

我这是套组合拳，用来达到千万级别的收益。

## 核心点

赚黑钱

因为正规做生意是很苦的。

### 游戏方面

我们需要把用户的钱，转化为游戏中的虚拟货币

再配上一些赌博技巧，如开宝箱，将其消耗。

### 把游戏当成“白手套”，黑产售卖

1. 淘宝上售卖游戏货币充值卡\(实际上就是一串字符\)，与真实货币1:1
2. 游戏有数据库，用来发卡和验证。数据库需要留存发出去的卡（以便验证）。并且得记录实际已兑换的金额。
3. 黑产这边也应该有数据库。用来，记录用户的账户、密码、余额。
4. 黑产不直接使用货币，它使用游戏货币充值卡。
